page web de to-do list

    ce projet consiste á creer une page web permettant de gérer des  tâches; c'est á dire 
    ajouter, afficher,marquer comme terminer ou supprimer des   tâches.
    La page web est développée avec HTML,CSS et javaScript.

Les fonctionnalités 
    -Ajouter une nouvelle tâche
    -Afficher la liste des tâches
    -Marquer une tâche comme terminée
    -Supprimer une tâche

Les fonctionnalités supplémentaires
    -bouton tout supprimer
    -sauvegarde dans le stockage local: les tâches sont enrigistrées dans lenavigateur (LocalStorage) pour qu'elles restent disponibles aprés un rechargement de la page
    -compteur de tâches:le nombre  total de tâches et le nombre de  tâches terminées s'affichent

Installation et utilisation
    1.cloner le projet 
    git clone https://github.com/Ndeye-Marieme-Fall/todolist.git
    cd todolist
    2.ouvrir l'application
    lancer index.html dans un navigateur pour commencer á utiliser l'application

Stucture du projet
    -index.html:contient la structure de la page ; la logique et les interactions avec les balises script
    -css.css:définit l'apparence et le design.
